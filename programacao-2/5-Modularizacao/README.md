# Questões para programar

# Métodos  1\. Crie um método que receba 2 números e retorne o maior valor. 

2\. Crie um método que receba 2 números e retorne o menor valor. 

3\. Crie um método que receba 3 números e retorne o maior valor. 

4\. Crie um método que receba 3 números e retorne o menor valor.  
					  
5\. Crie um aplicativo de conversão entre as temperaturas Celsius e Fahrenheit. Primeiro o usuário deve escolher se vai entrar com a temperatura em Celsius ou Fahrenheit, depois a conversão escolhida é realizada através de um comando SWITCH. Não esqueça de criar um método que receberá os parâmetros necessários para realizar a conversão.  
					  
Se C é a temperatura em Celsius e F em fahrenheit, as fórmulas de conversão são:  
					  
**C= 5.(F-32)/9 F= (9.C/5) \+ 32**					

6\. Crie um método que receba 2 valores e retorne a sua soma. Se o valor da soma for negativo o método deverá retornar o valor 0\.

7\. Crie um método que receba 3 valores (a, b, c) e retorne verdadeiro se a\>b\>c e falso em caso contrário.

8\. Crie um método que receba três valores (ano, mês, dia) e verifique se é uma data válida. O ano deverá estar entre 1900 e o presente ano. Deverá retornar uma mensagem de válido ou inválido.

9\. Faça um programa para imprimir:  
    1  
    1   2  
    1   2   3  
    .....

    1   2   3   ...  n  
Dado um valor **n** informado pelo usuário. Escreva um método que receba um valor **n** e imprima até a n-ésima linha.
